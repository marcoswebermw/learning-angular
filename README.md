## Angular

É open source.  
Mantido pela Google em parceria com a Microsoft(TypeScript).  
Usa SPA(Single Page Application).  

### Blocos principais do Angular

* Componentes;
* Data Binding;
* Roteamento;
* Diretivas;
* Serviços;
* Template;
* Metadata;
* Injeção de Dependência;
